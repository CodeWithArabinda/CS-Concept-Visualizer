Why Linked Lists?

Definition: A collection of nodes where each node contains data and a pointer to the next node.

Dynamic Size: Unlike arrays, linked lists can grow or shrink in memory without needing to be reallocated.

Analogy: Think of a scavenger hunt or a train. Each clue (or carriage) leads you to the next one!
