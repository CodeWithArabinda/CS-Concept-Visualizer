Kosaraju's Algorithm

  1. Phase 1: Perform a standard DFS. Push nodes to a stack only after fully exploring all their branches (finish time).
  2. Phase 2: Reverse the direction of all edges in the graph (Transpose).
  3. Phase 3: Pop nodes from the stack. If unvisited, start a new DFS on the reversed graph to identify one Strongly Connected Component.
