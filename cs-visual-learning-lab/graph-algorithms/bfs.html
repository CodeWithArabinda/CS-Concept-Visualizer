About BFS

Breadth-First Search explores the graph level by level. It visits the starting node, then all its direct neighbors, then the neighbors of those neighbors, and so on.

It uses a Queue (First-In-First-Out) to keep track of nodes to visit next, ensuring nodes closer to the start are processed before nodes further away.
