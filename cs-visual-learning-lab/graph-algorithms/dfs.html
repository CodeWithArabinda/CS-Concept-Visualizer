About DFS

Depth-First Search explores as far as possible along each branch before backtracking.

It uses a Stack (Last-In-First-Out) mechanism. When a node has no unvisited neighbors left, it is popped off the stack, and the algorithm backtracks to the previous node to check for unexplored paths.
