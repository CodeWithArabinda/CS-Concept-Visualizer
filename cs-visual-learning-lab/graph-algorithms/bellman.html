Algorithm Theory

Edge Relaxation

For an edge (u → v) with weight w, if dist[u] + w < dist[v], we update dist[v] to this new, shorter distance.

Repeated Passes

Unlike Dijkstra's, Bellman-Ford relaxes all edges V-1 times (where V is the number of vertices). This guarantees the absolute shortest path even with negative weights.

Negative Cycles

After V-1 passes, if we can still relax an edge, it means the graph contains a negative weight cycle (a loop that infinitely decreases the path distance).

Time Complexity O(V × E)
