Algorithm Theory

1. Sort Edges

Kruskal's is a Greedy Algorithm. It begins by sorting all edges in the graph in ascending order of their weight.

2. Cycle Detection (Union-Find)

It evaluates edges one by one. To prevent cycles, it uses a Disjoint Set (Union-Find) data structure.

  • Find: Checks if two nodes are already in the same tree.
  • Union: Merges two trees if an edge connects them.

Complexity Metrics

Time Complexity O(E log E)
Space Complexity O(V + E)
