Algorithm Theory

The Expanding Tree

Unlike Kruskal's, which sorts all edges globally, Prim's algorithm is a Greedy Algorithm that builds the MST from a single starting node outward.

The Frontier

At each step, it evaluates the frontier: all edges connecting a node currently in the MST to a node outside it. It always picks the smallest weight edge to add.
