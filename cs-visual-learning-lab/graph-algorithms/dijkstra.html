What is Dijkstra's Algorithm?

It is a greedy algorithm used to find the shortest path between nodes in a graph. It was conceived by computer scientist Edsger W. Dijkstra in 1956.

Greedy Choice: At each step, it picks the unvisited node with the smallest distance value. It never revisits nodes, which is why it requires non-negative weights.

Real-world Applications:

  • GPS Navigation (finding quickest routes)
  • Network Routing Protocols (OSPF)
  • Digital Circuit Layout
  • Robotic Path Planning
