Counting Sort is a non-comparison sorting algorithm. Instead of comparing elements, it counts the occurrences of each unique element in the array.

Limitations: It is only efficient when the range of potential items (k) is relatively small compared to the number of items (n). It only works with integers.

  • Time Complexity O(n + k)
  • Space Complexity O(n + k)
  • Stable Sort? Yes
