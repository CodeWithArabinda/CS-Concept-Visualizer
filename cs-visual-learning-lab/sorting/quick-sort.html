Quick Sort Strategy

quickSort(arr, low, high):
  if low < high:
    p = partition(arr, low, high)
    quickSort(arr, low, p - 1)
    quickSort(arr, p + 1, high)
partition(arr, low, high):
  pivot = arr[high]
  i = low - 1
  for j = low to high - 1:
    if arr[j] < pivot: swap(i, j)
  swap(i + 1, high)

Educational Summary

Quick Sort is a highly efficient sorting algorithm. It uses the Lomuto Partition Scheme in this visualization, which picks the last element as the pivot.

Elements smaller than the pivot move to the left, and larger ones move to the right. This "Divide and Conquer" approach ensures that once a pivot is placed, its position is permanent.

  • Partitioning: The core engine that reorders elements around a pivot.
  • Recursive: The algorithm calls itself on the left and right subarrays until base cases (size 1 or 0) are reached.
  • In-Place: It requires very little extra memory compared to Merge Sort.
