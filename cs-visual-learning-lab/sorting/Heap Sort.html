Educational Panel

Heap Sort converts the array into a Max Heap (a complete binary tree where every parent is greater than its children).

It then repeatedly extracts the maximum element (the root), swaps it with the last element of the heap, and runs Heapify to restore the heap property for the remaining elements.

  • Time Complexity O(n log n)
  • Space Complexity O(1)
