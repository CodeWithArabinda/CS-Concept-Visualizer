Educational Panel

Insertion Sort builds the sorted array one element at a time. It iterates through the input elements, growing a sorted list behind it. At each iteration, it removes one element from the input, finds the location it belongs within the sorted list, and inserts it there.

  • Best Case O(n)
  • Average Case O(n²)
  • Worst Case O(n²)
  • Space Complexity O(1)
