How it works

Selection Sort works by dividing the array into two parts: a sorted portion at the front and an unsorted portion at the back.

In each iteration, the algorithm finds the minimum element in the unsorted portion and swaps it with the first element of that portion.

  • Minimum: We keep track of the smallest value found so far in the current pass.
  • Swap: Once the pass is done, one swap puts the minimum in its final place.
  • In-Place: No extra memory is needed, but the complexity is always O(n²).
