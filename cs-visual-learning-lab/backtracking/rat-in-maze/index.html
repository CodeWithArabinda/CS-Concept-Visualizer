Learning: Backtracking

Backtracking is a recursive strategy for solving constraint satisfaction problems. In this maze, the "Rat" explores a path as far as possible. If it hits a dead end (a wall or a path already visited), it backtracks to the last valid cell and tries a different direction.

Base Cases

  • Goal reached (Success)
  • Out of bounds (Failure)
  • Hit a wall (Failure)

Directions

The algorithm prioritizes moving Down and Right to reach the bottom-right destination efficiently.
