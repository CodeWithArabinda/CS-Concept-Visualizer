Algorithm Theory

The Decision Tree

The Subset Sum problem uses backtracking to explore a binary decision tree. For every element in the array, the algorithm makes a choice:

  • Include it in the current sum (Left Branch).
  • Exclude it from the current sum (Right Branch).

Pruning (Optimizing)

To avoid checking all 2n combinations, we prune branches early if:

  • The cumulative sum exceeds the target (assuming positive numbers).
  • We run out of elements and the sum != target.
Time Complexity O(2n)
Space Complexity O(n)
