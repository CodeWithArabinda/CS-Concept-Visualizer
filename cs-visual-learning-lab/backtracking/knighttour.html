About the Algorithm

The Knight's Tour is a classic mathematical problem. The knight must visit every square exactly once using only valid "L-shaped" chess moves.

This visualizer uses Depth-First Search (DFS) with Backtracking. If the knight reaches a dead end before visiting all squares, it backtracks (undoes its recent moves) to try alternative paths.

Note: To ensure the visualization finishes in a reasonable time, it uses Warnsdorff's heuristic, which prioritizes moves that have the fewest onward choices, vastly reducing the need to backtrack on larger boards.
