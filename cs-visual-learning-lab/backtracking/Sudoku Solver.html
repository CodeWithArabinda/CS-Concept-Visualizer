Backtracking Theory

What is Backtracking?

Backtracking is an algorithmic technique for solving problems recursively by trying to build a solution incrementally, one piece at a time. It removes (backtracks) solutions that fail to satisfy the constraints of the problem at any point in time.

Sudoku Constraints

Before placing a number (1-9) in an empty cell, the algorithm checks three constraints:

  • Row must not contain the number.
  • Column must not contain the number.
  • The 3x3 sub-grid must not contain the number.

Complexity Metrics

Time Complexity O(9n)
Space Complexity O(n)

* Where n is the number of empty cells. Recursion depth reaches at most 81.
