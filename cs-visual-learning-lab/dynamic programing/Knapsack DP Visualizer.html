DP State Relation

For every item i and capacity w, we make a decision:

dp[i][w] = max(
  dp[i-1][w], // Exclude
  val[i] + dp[i-1][w-wt[i]] // Include
)
  • Exclude: We don't take the item. The max value is the same as if we only considered the previous items.
  • Include: If the item fits, we add its value to the optimal solution for the remaining capacity.
