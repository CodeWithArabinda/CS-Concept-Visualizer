Algorithm Theory

The Goal

Find the minimum number of operations required to transform String A into String B using only Insertions, Deletions, and Replacements.

2D DP Table

We build a table where dp[i][j] represents the edit distance between the first i characters of String A and the first j characters of String B.

Operations & Dependencies

  • Insert: Looks left to dp[i][j-1]. We conceptually insert B[j] to match.
  • Delete: Looks up to dp[i-1][j]. We conceptually delete A[i] to match.
  • Replace/Match: Looks diagonally up-left to dp[i-1][j-1]. We replace A[i] with B[j], or do nothing if they match.
Time Complexity O(m × n)
Space Complexity O(m × n)
