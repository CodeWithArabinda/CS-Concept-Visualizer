let dp = new Array(n).fill(1); for (let i = 1; i < n; i++) { for (let j = 0; j < i; j++) { if (arr[i] > arr[j] && dp[i] < dp[j] + 1) { dp[i] = dp[j] + 1; } } }

Algorithm Theory

The Subproblem

We want to find the Longest Increasing Subsequence (LIS) ending exactly at index i.

Initially, every element is an increasing subsequence of length 1 (itself). So, dp[i] = 1 for all i.

Building the Sequence

For each element at index i, we look back at all previous elements at index j (where j < i).

  • If arr[i] > arr[j], we can append arr[i] to the subsequence ending at j.
  • The new length becomes dp[j] + 1.
  • We update dp[i] if this new length is greater than its current value.
Time Complexity O(n²)
Space Complexity O(n)

Note: A more advanced O(n log n) approach exists using Binary Search.
