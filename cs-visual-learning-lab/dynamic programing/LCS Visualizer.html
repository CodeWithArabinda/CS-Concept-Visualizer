String Matching Logic

We compare substrings of A and B. For each cell dp[i][j], we look at A[i-1] and B[j-1]:

// If characters match:
if (A[i-1] == B[j-1]) {
  dp[i][j] = dp[i-1][j-1] + 1;
}
// If characters do not match:
else {
  dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
}
  • Match: We extend the longest common subsequence found so far (from the diagonal).
  • No Match: We carry forward the best sequence found either by ignoring the current char of A (top) or B (left).
