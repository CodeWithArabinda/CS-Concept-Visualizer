Educational Panel

Linear Search sequentially checks each element of the list until a match is found or the whole list has been searched.

It is the simplest search algorithm and does not require the array to be sorted beforehand.

  • Best Case O(1)
  • Worst Case O(n)
  • Average Case O(n)
