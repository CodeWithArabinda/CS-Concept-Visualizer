Algorithm Theory

Exponential Search

Exponential search is particularly useful for unbounded or infinite arrays, and finding elements closer to the beginning of a sorted array quickly.

Two-Phase Strategy

  1. Range Expansion: Start at index 1 and repeatedly double the index (1 → 2 → 4 → 8...) until the value at the current index exceeds the target.
  2. Binary Search: Once the bounds are found, perform a standard Binary Search within the discovered sub-array [i/2, min(i, n)].

Complexity Metrics

Time Complexity O(log i)
Worst Case O(log n)
Space Complexity O(1)

Note: i is the index of the element being searched. This makes it faster than Binary Search O(log n) if the element is near the beginning.
