Algorithm Theory

What is Jump Search?

Jump Search is a searching algorithm for sorted arrays. Instead of checking every element (Linear Search) or halving the search space (Binary Search), it jumps ahead by fixed steps to skip elements.

The Strategy

  1. Calculate the optimal jump size: √n
  2. Jump through the array by this size until you find a value greater than the target.
  3. This determines the block where the target might exist.
  4. Perform a standard linear search backwards or forwards within this specific block.

Complexity Metrics

Time Complexity O(√n)
Optimal Jump Size √n
Space Complexity O(1)

Jump search falls between Linear Search O(n) and Binary Search O(log n) in terms of speed, but is particularly useful when reversing through an array is costly.
